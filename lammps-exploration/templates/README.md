# LAMMPS Exploration Templates

These templates are teaching scaffolds for LAMMPS exploration in DP-GEN. They are intentionally incomplete.

## Files

| File | Purpose |
|---|---|
| `input.lammps.template` | LAMMPS input-script skeleton for DeePMD/DP-GEN exploration. |
| `input.plumed.template` | PLUMED input skeleton for collective variables, restraints, and diagnostics. |
| `reference-examples/` | Place redacted real LAMMPS scripts here when available. |

Replace every `TODO_USER_APPROVAL` token before running. Confirm syntax against official LAMMPS and DeePMD-kit docs for the installed versions.

## Real Reference Example

`reference-examples/corr-lammps-plumed/` contains real LAMMPS and PLUMED inputs from a CORR-related DP-GEN workflow. Use it as a style/reference example, not as H2/graphene defaults.
